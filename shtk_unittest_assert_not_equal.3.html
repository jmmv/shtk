NAME

shtk_unittest_assert_not_equal — Fatal inequality check between two values

LIBRARY

shtk_import unittest

SYNOPSIS

shtk_unittest_assert_not_equal unexpected_value actual_value

DESCRIPTION

The shtk_unittest_assert_not_equal function implements an inequality check between two values. If actual_value does not match the value given in unexpected_value (usually hardcoded in the test), then the check is considered a success. Otherwise, the check fails and the test case is aborted.

SEE ALSO

shtk(3), shtk_unittest(3)

HISTORY

shtk_unittest_assert_not_equal first appeared in shtk 1.6.
